# 🏨 Hotel Booking Cancellation Analysis

## 📌 Project Overview

Hotel booking cancellations can significantly impact revenue, occupancy planning, and overall business operations. This project analyzes hotel booking data using Microsoft Excel to identify cancellation patterns, customer behavior, and key factors influencing booking cancellations.

The analysis provides actionable insights that can help hotel management improve decision-making, optimize booking strategies, and reduce cancellation rates.

## 🎯 Objectives

* Analyze hotel booking cancellation trends.
* Identify factors influencing booking cancellations.
* Understand customer booking behavior.
* Create visual reports and dashboards for data-driven insights.
* Support business decision-making through data analysis.

## 🛠️ Tools Used

* Microsoft Excel
* Pivot Tables
* Excel Functions & Formulas
* Charts and Graphs
* Data Cleaning Techniques
* Filtering and Sorting

## 📊 Dataset Information

The dataset contains hotel booking records with attributes such as:

* Booking Status
* Lead Time
* Number of Adults and Children
* Meal Type
* Market Segment
* Customer Type
* Previous Cancellations
* Special Requests
* Reservation Details

These features were analyzed to identify booking trends and cancellation patterns.
